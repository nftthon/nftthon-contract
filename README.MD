## Installation

### 1. Clone repo

### 2. Add your Open AI API Key to .env.local
```
cd app
echo OPEN_AI_API_KEY=<Your Open AI API Key> > .env.local
more .env.local 
```
### 3. Launch App
```
cd app
yarn dev
```
### 4. Launch Microsoft Edge or Chrome, and access https://localhost:3000 :+1:
